# Pomodoro Timer

A beautiful, full-featured Pomodoro Timer application built with vanilla HTML, CSS, and JavaScript.

## Features

- 🎯 **Focus Mode**: Stay productive with customizable 25-minute focus sessions
- ⏰ **Break Modes**: Short breaks (5 min) and long breaks (15 min)
- 📊 **Session Tracking**: Visual progress indicators and session counters
- 💾 **Persistent State**: All settings and timer state saved to localStorage
- ⚙️ **Customizable**: Adjust focus, short break, and long break durations
- 🎨 **Modern UI**: Clean, responsive design with animated progress rings

## How to Use

Simply Ation to the "Settings" section to customize:
- Focus duration
- Short break duration
- Long break duration

These settings persist across sessions using localStorage.

## Browser Support

Works on all modern browsers that support:
- localStorage API
- ES6 JavaScript
- CSS animations
